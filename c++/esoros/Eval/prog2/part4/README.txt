  This program provides many ways to parse a text file that includes log files stored in the appache common log format.
 
 USEAGE: 
 run ./main <insert_filename.txt> all - This outputs all of the entires
 
 ./main <insert_filename.txt> bytes - This outputs the total number of bytes sent
 
 ./main <insert_filename.txt> host - This outputs all of the hosts
 
 To compile, run make in the root directory. 
 
 To compile the String test, run "make string_test"
 
 To compile the Logentry test, tun "make logentry_test"
